# Real-Time Webcam Filters

### Descrizione
Applicazione desktop interattiva in Python e OpenCV per l'elaborazione video in tempo reale da webcam. Consente l'applicazione dinamica di filtri colore avanzati, effetti di pixel art e tracciamento intelligente del volto tramite classificatori Haar Cascade per sfocatura dello sfondo ed elaborazioni localizzate.

### Requisiti di Sistema
- **Sistema Operativo**: Windows 10/11, macOS, o Linux (es. Raspberry Pi OS).
- **Hardware**: Qualsiasi webcam USB standard o fotocamera integrata.
- **Python**: Versione 3.8 o superiore.
